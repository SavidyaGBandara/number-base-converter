# Number Base Converter

A simple command-line program that converts numbers between bases 2 to 10.
This was made as part of my Python learning journey.

## How to Use

You will be prompted for:
1. The number you want to convert
2. The base of that number
3. The base you want to convert to

### Example

Enter the number to convert: 1010
Enter the base of the given number (e.g., 2, 8, 10): 2
Enter the base you want to convert to: 10
10

## Features

- Converts numbers between any base from 2 to 10
- Validates the input number against the given base
- Handles invalid base input without crashing
